## 最近の探求：RSIコンパス付き助走戦略の深化とDiscord通知システム構築 (2025年11月30日以降の活動)

### 1. バックテストのバグ修正と戦略開発の基盤固め
*   シグナル生成におけるルックアヘッドバイアス（未来のデータ参照）を排除し、正確なバックテストシミュレーション（高値・安値によるTP/SL判定）を実装。
*   プロジェクト内の`config.py`や各戦略ファイルにおけるパスの誤りを修正し、コードベースの堅牢性を向上。

### 2. 「RSIコンパス付き助走戦略」の確立
*   **コアアイデア:** 以下の組み合わせで「爆発的な動き」を捉える戦略を開発。
    *   **助走 (Running Start):** 値幅と出来高が連続して増加するパターン。
    *   **RSIコンパス:** 助走中のRSIレベルで方向性（上昇か下降か）を判断。
    *   **長期トレンドフィルター (SMA200):** 長期トレンドに沿った方向でのみトレードを行う。
*   **初期のウォークフォワード最適化結果 (バージョン1.0 - 固定TP/SL):**
    *   **総損益:** +39.16%
    *   **最大ドローダウン:** -20.19%
    *   **シャープレシオ:** 3.90
    *   **特徴:** 堅牢なプラスのエッジと管理可能なリスクを確認。

### 3. 動的パラメータ適応の試みと教訓
*   **ATRベースのTP/SL最適化を試行:** 市場のボラティリティに応じてTP/SLが変動する動的な戦略を目指す。
*   **結果:** 収益性がマイナスに転じ、ドローダウンが悪化。この戦略においてはATRベースのTP/SLが機能しないことを発見。これは、戦略の性質とATRの特性がミスマッチを起こしたためと結論。

### 4. シグナル生成への「ボラティリティフィルター」導入（バージョン1.1）
*   TP/SLではなく、シグナル自体を市場のボラティリティでフィルターするアプローチを検討。
*   **ATRベースのボラティリティフィルターを導入したウォークフォワード最適化結果 (バージョン1.1):**
    *   **総損益:** +9.53%
    *   **最大ドローダウン:** **-6.64%** (極めて低い水準)
    *   **シャープレシオ:** **23.05** (非常に高い水準)
    *   **総トレード数:** 13回 (極めて厳選されたシグナル)
    *   **特徴:** 驚異的なリスク管理能力と高いリスク調整後リターンを達成したが、トレード頻度が極端に低い「超厳選型」戦略となる。

### 5. ライブシグナル通知システム (`discord_notifier.py`) の構築
*   両戦略（バージョン1.0とバージョン1.1）の売買サインをリアルタイムでDiscordに通知するシステムを開発。
*   ライブデータ取得、インジケーター計算、シグナルチェック、Discord通知の機能を実装。
*   開発中に発生したスリープロジックのバグや`FutureWarning`を修正し、安定稼働を達成。
*   現在、`discord_notifier.py` はバックグラウンドで正常稼働中であり、市場環境が戦略の厳格な条件を満たすシグナルを待機している。

## Project Harmony 探求の記録：1時間足への次元上昇と「循環」哲学の実装 (2026年1月25日)

### 1. 【課題提起】日足グリッドの停滞と「罠」への疑念
*   **直面した課題:**
    *   日足（Daily）ベースの戦略は過去5年で安定していたが、直近1年（2025年）において利益率が約2%まで激減。「動いているのに稼げない」現象が発生。
*   **抱いた疑問:**
    *   「RSIなどのフィルター（罠）が厳しすぎて、今の相場のチャンスを逃しているのではないか？」
    *   「そもそも日足という時間軸が、現在の相場のスピードに合っていないのではないか？」
*   **試行と推移:**
    *   **RSI緩和実験:** 閾値を40→60へ、さらに撤廃（Pure Grid）へ。結果、取引数は増えたが、2022年のような「底なし沼」でポジションを抱え込みすぎる弱点が露呈。
*   **未踏・残された課題:**
    *   SMA以外のトレンドフィルター（ADXなど）による選別。

### 2. 【次元転換】1時間足へのダイブと「死」の体験
*   **新たな視点:**
    *   日足で見えない「微細な波（ノイズ）」こそが、今の相場の主食ではないか。
*   **試行と推移:**
    *   **1時間足ATRグリッド:** 時間軸を1Hへ落とし、ATR（ボラティリティ）に連動して網を張る実験。
    *   **ハイパーアグレッシブ設定（ATR 0.8倍）:** 利益の最大化を狙って網を密にした結果、過去5年のバックテストで **「破綻（資金ショート）」** を経験。
    *   **教訓:** 攻撃力（回転数）を上げると、防御力（資金管理）が追いつかなくなる物理的な限界点を確認。
*   **未踏・残された課題:**
    *   5分足や15分足での超高頻度取引（手数料負けのリスクが高いため今回は見送り）。

### 3. 【哲学的転回】「循環」の定義と統計的アプローチ
*   **課題:**
    *   破綻の原因である「高値掴みポジションの無限ナンピン（澱み）」をどう解消するか。
*   **視点（石川哲学）:**
    *   「時間や個数で制限するのではなく、エネルギー（価格）の自然な循環を促す」。
    *   損切りを「敗北」ではなく「新陳代謝」と再定義する。
*   **試行と推移:**
    *   **MAE（最大逆行幅）分析:** 5,000回以上のトレードを分析し、「ATR 7.0倍」までは耐えれば90%助かるが、それ以上は「死に至る病」であることを特定。
    *   **MFE（最大順行幅）分析:** 逆に「ATR 14.0倍」まで利益が伸びるポテンシャルがあることを発見。従来の「1倍利確」は宝の持ち腐れだった。
    *   **損切り貧乏の克服:** 当初、損切りを導入したらリターンがマイナスになったが、利確幅を大幅に伸ばす（1倍→14倍）ことで、損切りコストを吸収できるようになった。
*   **未踏・残された課題:**
    *   ポジションサイズ（枚数）を固定ではなく可変にする資金管理（ピラミッティングや逆マーチンゲール）。

### 4. 【統合】「呼吸するBot (Ishikawa Hybrid Model)」への進化
*   **課題:**
    *   「過去の激動」と「現在の静寂」の両方に対応できるパラメータが見つからない（固定値の限界）。
*   **解決策（順応）:**
    *   **「呼吸」の実装:** 利確幅（TP）を、現在のボラティリティ比率に応じて伸縮させる（相場が静かなら手前で、荒れたら遠くで）。
    *   **非対称な守り:** 損切り（SL）も変動させると暴落時に傷口を広げるため、あえて「固定」とし、堅牢な堤防とする。
*   **到達点:**
    *   **Timeframe:** 1H
    *   **Entry:** ATR 1.2（基本）〜 1.8（現在推奨）
    *   **TP:** Base 14.0 ATR（変動型）
    *   **SL:** Fixed 15.0 ATR（固定型）
    *   これにより、全期間で年利約20%を安定して叩き出すモデルが完成。
*   **未踏・残された課題:**
    *   半減期サイクルごとの「季節」を自動判定し、Entry/TP/SLのベース値を動的に切り替えるロジックの実装。

## 【深層解析】15分足『RSIコンパス付き助走戦略』の哲学と深化の可能性 (2026年1月29日)

### 1. 戦略の「なぜ (Why)」：3つの物理的仮説
39.16%の利益（Ver 1.0）を支えるのは、単なる数値の最適化ではなく、以下の3つの市場物理学的な仮説である。

*   **エネルギー保存の法則（助走 - Running Start）:** 爆発的変動の前には、値幅と出来高が連続して増加する「エネルギーの凝縮」が発生するという仮説。
*   **内部モメンタムの羅針盤（RSIコンパス）:** RSIを逆張りではなく「支配的な勢力がどちらにあるか」の順張り指標として定義。
*   **大河の流れ（SMA200フィルター）:** 短期的な爆発も長期トレンドという「大河」に逆らえば短命に終わる。「出る台」を絞り込む確率の壁。

### 2. 今後の精度向上（深化）への指針
*   **「助走」定義の物理的洗練:** 厳密な数値順から、回帰直線の傾きなど「勢い（モメンタム）」を多角的に捉えるロジックへの進化。
*   **「呼吸」するExit:** 石川モデルの哲学を継承し、エントリー時のATRに応じて動的にTP/SLを伸縮させる。

## 【極・深層解析】Project Harmony：無の深淵から呼吸する生命体への全記録 (2026年1月29日)

### 1. 原点：時間の概念のない世界 (The Timeless Void)
**「チャートに横軸（時間）は存在しない。あるのは縦軸（価格）の揺らぎだけだ」**
*   **初期思想:** 人間が作った「時間」という枠組みを排除し、純粋な価格変動（Tick）のみに反応するロジック (Omen V1) から始まった。
*   **物理的障壁:** 時間というフィルタを外した結果、無限のノイズ（ブラウン運動）すべてに反応。スプレッドと手数料という「摩擦」だけで資金が削り取られた。
*   **発見:** 時間とは、無意味なノイズを平滑化し、市場の合意（トレンド）を可視化するための「ローパスフィルタ」であるという逆説的な真実に到達。

### 2. グリッドの導入と「静的な網」の限界 (Static Grid)
**「カオスの中に秩序（網）を持ち込む」**
*   **日足の成功と停滞:** 過去5年は「強いトレンド」か「レンジ」で勝てたが、2025年の「動くがトレンドがない」相場で機能不全に陥る。
*   **RSIの罠:** 安全策（買われすぎで売らない）が、強い相場では「永遠に見ているだけ」の機会損失を生んだ。
*   **死の領域 (Dead Zone):** 逆にフィルタを外すと、下落トレンドで無限に買い下がり、資金が含み損の重力に捕らわれた。
*   **教訓:** 固定されたパラメータ（静的な網）は、相場の波長が変わった瞬間に無力化する。

### 3. 次元降下と「死」の体験 (Descent & Death)
**「重力を振り切るための加速」**
*   **実験:** 日足の鈍重さを嫌い、1時間足で極限まで回転数を上げるハイパーアグレッシブ設定（ATR 0.8倍）を試行。
*   **死のメカニズム:** 回転数増加＝ポジション数の指数関数的増加。急落時に資金回復が追いつかず、**物理的限界点（破産確率100%）**を体感。
*   **恐怖:** 物理法則（レバレッジの限界）は、いかなる希望的観測も許さないことを理解。

### 4. 循環の数理：MAE/MFE分析による解脱 (Enlightenment)
**「生き残るために、体の一部を切り捨てる」**
*   **哲学:** ポジションへの執着を捨て、データに「切るべき場所」を問う。
*   **MAE (最大逆行幅) の特異点:** 含み損が **ATR 7.0倍** を超えたポジションの生存率は10%未満。これを絶対的損切り（新陳代謝）ラインと定義。
*   **MFE (最大順行幅) の発見:** 利益は **ATR 14.0倍** まで伸びるポテンシャルがある。従来の薄利多売は利益の9割を捨てていた。
*   **革命:** 「損小利大 (Risk 7 : Reward 14)」という、グリッド戦略の常識を覆す構造の確立。

### 5. 最終形態：呼吸するBot (Ishikawa Hybrid Model)
**「相場の波長に同調する生命体」**
*   **非対称性の物理:**
    *   **攻めの呼吸 (Variable TP):** ATRが高い（相場のエネルギーが大きい）時は、利確幅を自動で広げて利益を最大化する。
    *   **守りの堤防 (Fixed SL):** どんな嵐でも損切り幅は広げない。**「利益は相場の都合（ATR）に任せるが、損失は自分の都合（資金管理）で決める」**という主導権の確保。
*   **1Hの必然性:** 原点の純粋さと日足の信頼性が交差する、最も効率的な「波の周波数」として1時間足を採用。
